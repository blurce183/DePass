# DePass - Secure Password Generator

DePass is a lightweight, high-security password generator that runs entirely in your browser. It uses cryptographic-grade entropy to ensure your passwords are as secure as possible.

## 🔑 Key Features

- **Privacy First:** No data is sent to any server. Everything happens locally on your device.
- **Session History:** Keeps track of your last generated passwords during the current session.
